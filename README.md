# Algorithms Study

Implementações e anotações de estudo baseadas no livro **Entendendo Algoritmos** (Grokking Algorithms), escritas em TypeScript.

## Objetivo

Acompanhar a leitura do livro implementando os algoritmos em código e documentando os conceitos aprendidos em cada capítulo.

## Estrutura

```
src/          # Implementações dos algoritmos
docs/         # Anotações e explicações por capítulo
```

## Capítulos

| Capítulo | Algoritmos | Docs |
|----------|-----------|------|
| 01 — Introdução a algoritmos | [Busca Binária](src/chapter-01/binarySearch.ts) | [Busca Binária](docs/chapter-01/binary-search.md) · [Notação Big-O](docs/chapter-01/big-o-notation.md) · [Exercícios](docs/chapter-01/exercises.md) |

## Como executar

```bash
npm install

# Capítulo 01
npm run run:ch01
```

## Stack

- TypeScript
- [tsx](https://github.com/privatenumber/tsx) para execução direta de `.ts`
